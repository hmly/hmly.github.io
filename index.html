Projects Descriptions

SQL Sandbox

Users can write limited queries to manipulate and modify an existing, populated relational database where the results are immediately displayed.

Automated Product Extraction

Automates the extraction of product-attributed words from a file of product descriptions through the use of graph analysis and clustering.

Finite Mathematics Website

Organize the lecture materials for the course; include texts, videos, comment sections, and interactive exercises.

Data Science Projects

Team Projects: cas-faculty, dmo-analysis, regr-models, usda-nutrients, yelp-reviews

Weather-Forecast

Show the current temperature, pressure, humidity, visibility, wind speed, and a week-long forecast.

Java GUI Apps

Java Graphical User Interface Apps: Text Editor, Text Clock, and Calculator

Web Server

An Erlang web server that supports the fundamental HTTP methods: GET, POST, PUT, and DELETE.

LS Routing Simulation

Simulate a network of packet-switching programs; IP routers that forwards datagrams over the Internet.

Python Web APIs Apps

Apps utilizing web APIs: ip_lookup, quote_tracker, weather_forecast, whois_search, zipcode_lookup

JavaScript Calculator

A stylish calculator with key-binding and supports the basic mathematical operations and more.

Register Page

A register page with complex username, password, and email validation.

NLP Solutions

Solutions to most of the problems in the book, Natural Language Processing with Python.

SQLZoo Solutions

Solutions to all the practice exercises under the tab Tutorials: Learn SQL in stages.

Liang Solutions

Solutions to some of the problems in Introduction to Programming Using Python by Daniel Liang.

Project Euler Solutions

Solutions to a portion of the first 50 problems of Project Euler.

Digital Clock App

Display a digital clock and a timer which the user can use to set a time.

Favorite Websites App

Users can create and customize a list of their favorite websites.

Image Collage App

Display four distinct iconic landmarks around the world placed in a fashion resembling a collage.

Flickr Search App

Flickr query search with the feature to save a query to a list.

Math Quiz App

A math quiz with three randomly generated expressions.

Flag Quiz App

A quiz where the user has to identify the country associated with a certain flag.

Target Heart Rate Calculator App

Calculates the target heart rate range and max heart rate.

Mortgage Calculator App

Calculates the monthly payment for a mortgage plan given the purchase price and other related info.

Temperature Converter App

A Fahrenheit (F), Celsius (C), and Kelvin (K) temperature conversion.

U.S. States Quiz App

A ten questions quiz with the score and other statistics displayed at the end.

Travel Cost App

Calculate the cost of travel given the distance, mpg, and gas price.

Cities Questions App

Users can customize a list of their favorite US cities from a predefined list.
